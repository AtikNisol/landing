# Мультимедийный лонгрид: битмейкинг (Ableton)

Тёмная, вдохновляющая веб‑страница для широкой публики о битмейкинге: история, возможности, практика, тренды. Акцент на Ableton Live. Все медиа — локальные (PNG/JPG, MP3/WAV, MP4).

## Структура

- `index.html` — страница лонгрида
- `styles/main.css` — стили (тёмная тема, фиолетовый акцент, секционные цвета)
- `scripts/main.js` — интерактив (липкое оглавление, до/после, галерея)
- `assets/images/` — изображения (PNG/JPG)
- `assets/audio/` — аудио (MP3/WAV)
- `assets/video/` — видео (MP4)
- `assets/media-manifest.json` — список ожидаемых медиа (названия и назначение)

## Ожидаемые локальные медиа

См. `assets/media-manifest.json` для имён файлов и мест в странице. Вы можете менять имена/пути, но тогда обновите ссылки в `index.html`.

## Как открыть локально (Windows)

Просто дважды кликните `index.html` или запустите локальный сервер:

```powershell
# В папке longread
powershell -NoProfile -ExecutionPolicy Bypass -Command "Start-Process msedge index.html"
```

Либо (Node.js):

```bash
npx serve .
```

## Деплой на GitHub Pages

1. Создайте репозиторий и добавьте файлы папки `longread` в корень.
2. Закоммитьте и запушьте.
3. В настройках репозитория включите Pages: `Deploy from a branch` → `main` → `/` (root).
4. Откройте выданный URL.

## Настройка тем и акцентов

- Основной акцент: фиолетовый (`--accent`).
- Для секций используются мягкие тёмные градиенты; можно править переменные в `:root`.

## Доступность и производительность

- Видео без звука в херо: `muted loop autoplay playsinline`.
- `prefers-reduced-motion`: отключает автозапуск.
- Картинки — используйте разумные размеры; для скриншотов — 1600px по ширине достаточно.
- Аудио — фрагменты 15–40 секунд, 128–192 kbps MP3 (или WAV, но учитывайте размер).
- Видео — 720p H.264 MP4, до 30–60 секунд.

## Разделы

- Истоки — таймлайн + фото/видео/аудио.
- Ableton — возможности, сетка медиа, цитата с юмором.
- Практика — пошаговый воркфлоу, видео/аудио/картинка.
- Звук — блок до/после с двумя MP3.
- Тренды — простая инфографика + аудио‑коллаж.
- Сетап — галерея (3 PNG).
- Сравнение — короткий текст + видео.
- Ресурсы — список.
- Заключение — вдохновляющий финал.

## Кастомизация

- Замените плейсхолдеры в `index.html` на свои файлы.
- Можете менять секционные цвета в `styles/main.css`.
- Добавьте новые разделы и ссылки в оглавление.

## Лицензия медиа

Убедитесь, что используете медиа, права на которые у вас есть. Не загружайте защищённый контент без разрешения.
